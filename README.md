# 电脑商城(期末小项目)

## 具体介绍

> springboot + vue + mysql的项目。
>
> **前后端不分离**：可以理解为纯前端项目，一个写个小组作业实在不容易。

## 具体功能

> 1. 用户的登录注册
> 2. 网页的跳转
> 3. 个人消息的管理等
> 4. 沙箱模拟支付功能。
